PROJECT TITLE
Temperature Sensor-Based LED Control using Arduino

OBJECTIVE
To simulate a sensor-based IoT system using Tinkercad without requiring physical hardware.

DESCRIPTION
This project demonstrates a simple IoT simulation in which an LED is controlled based on the temperature detected by a temperature sensor. The circuit was designed and tested using Tinkercad, allowing real-time simulation of sensor behavior and Arduino responses.

COMPONENTS USED
Arduino Uno
Temperature Sensor (TMP36)
LED
220Ω Resistor
Breadboard
Jumper Wires
Software Used
Tinkercad
Arduino IDE

WORKING PRINCIPLE
The Arduino continuously reads the temperature value from the sensor. When the temperature exceeds the predefined threshold, the LED turns ON. When the temperature falls below the threshold, the LED turns OFF.
