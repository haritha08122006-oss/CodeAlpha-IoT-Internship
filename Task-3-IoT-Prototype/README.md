Task 3 – IoT Prototype

PROJECT TITLE
Motion Detection Alarm using Ultrasonic Sensor

OBJECTIVE
To develop an IoT-based motion detection alarm using an Arduino Uno and an HC-SR04 ultrasonic sensor. The system detects nearby objects and activates an LED and buzzer when an object is within a predefined distance.

COMPONENTS USED
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- LED
- 220Ω Resistor
- Piezo Buzzer
- Jumper Wires

WORKING PRINCIPLE
The ultrasonic sensor continuously measures the distance to nearby objects. When an object is detected within 20 cm, the Arduino turns ON the LED and activates the buzzer. If the object moves beyond the threshold distance, both the LED and buzzer turn OFF.
